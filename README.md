# Simulación de Potencial Eléctrico: Anillo y Cilindro

Este proyecto grafica el campo de potencial eléctrico generado por un anillo cargado con una carga en el centro situado dentro de un cilindro conectado a tierra. 
Es un problema clásico de electromagnetismo resuelto analíticamente. Abierto a futuras mejoras, cálculo numérico, diferentes visualizaciones, etc. 

- **Visualización 2D:** Mapas de color.
- **Visualización 3D:** Distribución de cargas.
